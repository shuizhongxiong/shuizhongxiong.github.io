eclipse 常用快捷键

发表于 2018-06-12 | 更新于 2018-06-13 | 分类于 工具

1、代码格式化：control+shift+f

2、快速查找资源： control+shift+r

3、查找和替换： control+f

4、删除一整行： control+d

5、代码提示： alt+/

6、显示大纲： control+o

7、回退： control+z

8、显示属性： alt+enter

9、重命名：f2

10、跳到指定行：control+l

11、选中行上下移：control+方向键

12、查找下一个相同的： control+k

13、在右边显示出当前打开的所有文件：Ctrl + E

14、Ctrl + Pg Up ,Ctrl + Pg Dn ：逐个文件跳跃

15、Ctrl + O：可以显示当前文件的所有属性和方法

16、Ctrl + H： 进行全局搜索和局部搜索

17、Alt + Shift + N：创建各种类型的文件

18、Ctrl + Shift + P：左右括号跳转

19、tab ：缩进

20、Shift + tab：反缩进

21、Ctrl + Shift + T：查找源

22、Alt + ↑ / ↓：整行上移或下移

23、control+Q ：直接跳到上次编辑的地方

24、control+H：全局查找

25、alt+左右方向键：多个编辑文件切换
